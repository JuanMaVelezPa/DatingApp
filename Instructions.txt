
DATABASE
    For update DB
        - cd API
        - dotnet ef migrations add [<Name new migration>]
        - dotnet ef database update
        - dotnet ef database drop
    For Table Photos
        - cd API
        - dotnet ef migrations add ExtendedUserEntitiy
            - For remove migration
            - dotnet ef migrations remove
        - dotnet ef database 
    Eliminate database
        - dotnet ef database drop
